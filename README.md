# hospital-Data

This project analyzes hospital and patient data using SQL.
The goal is to understand patient flow, medical expenses,
hospital performance, and department efficiency.
It uses real-world healthcare scenarios.



objective
Analyze hospital and patient data
Identify high-demand cities and departments
Study medical expenses and cost efficiency
Practice real-world SQL queries


Dataset Description
he dataset contains hospital records including patient details,
hospital name, department, city, medical expenses,
admission date, discharge date, and length of stay.
